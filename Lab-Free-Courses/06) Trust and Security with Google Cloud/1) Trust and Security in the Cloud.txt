1) Which cybersecurity threat demands a ransom payment from a victim to regain access to their files and systems.
ans) Ransomware

2) Which definition best describes a firewall?
ans) A network security device that monitors and controls incoming and outgoing network traffic based on predefined security rules

3) Which three essential aspects of cloud security form the foundation of the CIA triad?
ans) Confidentiality, integrity, and availability

4) Which is the responsibility of the cloud provider in a cloud security model?
ans) Maintaining the customer's infrastructure.

5) Which is a benefit of cloud security over traditional on-premises security?
ans) Increased scalability.

6) Which security principle advocates granting users only the access they need to perform their job responsibilities?
ans) Least privilege

7) What common cybersecurity threat involves tricking users into revealing sensitive information or performing actions that compromise security?
ans) Phishing

8) Which cloud security principle ensures that security practices and measures align with established standards and guidelines?
ans) Compliance

9) Which cybersecurity threat occurs when errors arise during the setup of resources, inadvertently exposing sensitive data and systems to unauthorized access?
ans) Configuration mishaps

10) Which cloud security principle relates to keeping data accurate and trustworthy?
ans) Integrity
